# Alpha

## Project

This project use the [NetCore vue-spa template](https://github.com/NetCoreTemplates/vue-spa)

> .NET Core 2.1, Vue CLI 3, TS

### Project setup

From the `Alpha` dir

```bash
# install dependencies
$ npm install

# Run project
$ npm run serve
$ dotnet run

# Production build
$ npm run build

# Deployments
$ npm run publish
```
